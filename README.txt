How to install dependencies for server

Server has 2 dependicies, Python and R

To install Python dependencies
- First go to https://www.python.org/downloads/
- Download Python
- Add Python to path during installation (tick the related checkbox)
- Go to server and run install_lib.py by double clicking it

To install R dependencies
- First go to https://cran.r-project.org/bin/windows/base/
- Download R
- Add R to path during installation (tick the related checkbox)
- run command prompt
- run command 'r'
- run command install.packages("rgdal")
- run command install.packages("raster")
- run command install.packages("plotrix")
- run command install.packages("stringr")
